# CyP-Estudio-Juridico

Sitio estático inicial para el estudio con dos vistas:

- `index.html`: landing pública del estudio jurídico
- `expediente.html`: panel interno para gestión de expedientes

Archivos principales:

- `styles.css`: estilos compartidos
- `script.js`: interacciones livianas para nav y FAQ

Uso local:

```bash
python3 -m http.server 8000
```
