# CyP-Estudio-Juridico

Ecosistema digital estático inicial para estudio jurídico premium en Argentina.

Pantallas incluidas:

- `index.html`: landing page institucional
- `portal.html`: login y dashboard de clientes
- `expediente.html`: vista de gestión de expedientes para abogados

Archivos compartidos:

- `styles.css`
- `script.js`
